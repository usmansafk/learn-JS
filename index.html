    What is DOM?

    When we write our HTML and view it in the browser, the browser turns it into a Document Object Model, or DOM. In the DOM, each element becomes an object or node- we can use built methods to interact with these nodes using JS...

    DOM General Concepts

  • DOM is similar to CSS - you select an element, and then you will decide what affect you want to apply to the element
  • You can assign an element to a variable for ease of use: `const element = document.getElementById('element')`
  • *document* is accessed through a global window object that is at your disposal. Within their are many useful properties and methods including a property named document
  • Once an element is selected, you get a node object or node list, which is similar to like array like object.
  • Some, but not all array methods can be used on a node list. However, the node list can also be easily converted to an array list.

    • Window AND Document Overview

  • window object - browser api - the browser giving methods and properties for us to work with the browser

  • document - within the window object we have the document object.

  • **console.dir -** allows you to see all the properties and methods available for the node object you have selected `console.dir(document);` In this case, all the properties and methods on the document objectt that can be accessed
